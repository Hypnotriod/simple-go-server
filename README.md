# simple-go-server
Simple TCP test server written in `go`, configured for `VSCode`  

## Lauch (Go plugin required)
`VSCode` -> `Ctrl + Shift + B`:  
**Debug Stdin** - to lauch session with stdin  
`VSCode` -> `Ctrl + Shift + D`:  
**Attach Debug** - to start debugging  
`Putty` -> `Raw` at `127.0.0.1` host and `9876` port
